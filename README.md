# ✅ JS Validation

A simple and responsive **form validation** project built with pure HTML, CSS, and JavaScript — with real-time feedback and clean UI.

---

## 🚀 Live Demo

> Open `index.html` directly in your browser — no setup needed.

---

## ✨ Features

- 🎨 **Clean UI** — Styled with custom CSS and Font Awesome icons
- 📱 **Responsive Design** — Works on all screen sizes
- ⚡ **Zero Dependencies** — Pure Vanilla JavaScript, no libraries

---

## 🗂️ Project Structure

```
js-validation/
│
├── index.html        # Main form page
├── css/              # Stylesheets
├── js/               # Validation logic
└── webfonts/         # Font Awesome icons
```

---

## 🛠️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/omarkhattab74/js-validation.git
```

### 2. Open in browser

```bash
cd js-validation
open index.html
```

> No build tools or npm required. Just open `index.html` in any modern browser.
